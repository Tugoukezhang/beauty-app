## 一、项目概述

### 1.1 产品定位

| 项目 | 说明 |
|------|------|
| **产品名称** | i妆 - 美妆教学小程序 |
| **产品类型** | 美妆在线教育平台（微信小程序） |
| **核心价值** | 连接化妆老师与学员，提供专业美妆课程内容 |
| **目标用户** | 美妆爱好者、备婚新娘、化妆师进修人群 |
| **平台载体** | 微信小程序（优先）、APP（后续） |

### 1.2 产品愿景

> **使命**：让每个人都能学会专业的美妆技巧
>
> **愿景**：成为备婚新娘和美妆爱好者首选的学习平台
>
> **价值观**：专业、真实、用心

### 1.3 核心业务逻辑

```
┌─────────────────────────────────────────────────────────────┐
│                        i妆 业务闭环                          │
│                                                             │
│    ┌──────────┐    ┌──────────┐    ┌──────────┐            │

```

**审核规则：**

| 类型 | 内容示例 | 处理方式 |
|------|----------|----------|
| 敏感词 | 政治/色情/违禁词 | 直接拦截 |
| 广告推广 | 联系方式/二维码 | 先审后发 |
| 抄袭搬运 | 举报核实后处理 | 下架+警告 |
| 低俗内容 | 暴露/暗示内容 | 人工复核 |

**内容举报功能：**

| 举报类型 | 说明 |
|----------|------|
| 广告推广 | 硬广/软广/二维码 |
| 低俗内容 | 暴露/暗示内容 |
| 抄袭搬运 | 原创声明被侵犯 |
