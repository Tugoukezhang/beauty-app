## 🎯 **这个项目是做什么的？**

### **核心功能一句话概括**
**这是一个让用户通过个人微信与本地Claude Code进行对话的桥接工具。**

### **详细解释**
想象一下这样一个场景：
- 你正在手机上用微信和朋友聊天
- 突然有个编程问题或技术问题需要解决
- 你可以直接在微信中@一个特殊的机器人
- 这个机器人会连接到你电脑上运行的Claude Code（一个基于Anthropic Claude模型的AI编程助手）
- Claude Code会分析你的问题并给出答案，通过微信机器人回复给你

### **类比理解**
这就像是：
- **微信版Copilot** - 但针对Claude模型
- **移动端AI助手** - 通过微信这个最常用的App访问AI能力
- **远程编程助手** - 即使不在电脑前也能获得编程帮助

## 🔧 **技术架构**

### **系统架构图**
```
手机微信 ↔ ilink bot API ↔ Node.js守护进程 ↔ Claude Code SDK ↔ 本地Claude模型
```

### **技术栈**
- **前端/接口**: TypeScript + Node.js
- **AI模型**: Claude Code (基于Anthropic Claude)
- **通信协议**: ilink bot API (微信机器人接口)
- **平台支持**: macOS (launchd), Linux (systemd/nohup)
- **数据存储**: 本地SQLite/JSON文件

### **核心组件**
1. **微信连接层** - 处理微信消息收发
2. **消息转发层** - 在微信和Claude之间转发消息
3. **权限管理层** - 控制Claude的工具使用权限
4. **会话管理层** - 维护对话上下文
5. **守护进程层** - 确保服务持续运行

## 🚀 **主要功能特性**

### **1. 基础对话功能**
- ✅ **文字聊天** - 在微信中直接与Claude对话
- ✅ **图片识别** - 发送图片，Claude会分析内容
- ✅ **实时进度** - 显示Claude的思考过程（🔧, 📖, 🔍等图标）
- ✅ **会话持久化** - 保持对话上下文

### **2. 高级控制功能**
- ✅ **权限审批** - 当Claude需要执行操作时请求批准
- ✅ **思考预览** - 在执行前显示Claude的推理过程（💭图标）
- ✅ **中断支持** - 可随时发送新消息中断当前任务
- ✅ **模型切换** - 支持不同的Claude模型

### **3. 命令系统**
```bash
/help          # 显示帮助信息
/clear         # 清除当前会话
/model <名称>  # 切换Claude模型
/prompt [文本] # 查看或设置系统提示词
/status        # 查看当前会话状态
/skills        # 列出已安装技能
```

### **4. 权限模式**
- **default** - 手动审批每个工具使用
- **acceptEdits** - 自动批准文件编辑
- **plan** - 只读模式，禁止工具使用
- **auto** - 自动批准所有工具（危险模式）

## 📦 **安装要求**

### **硬件/软件要求**
- **操作系统**: macOS 或 Linux (不支持Windows)
- **Node.js**: 版本 >= 18
- **Claude Code**: 已安装 `@anthropic-ai/claude-agent-sdk`
- **微信账号**: 个人微信，需要扫码绑定

### **安装步骤**
```bash
# 1. 克隆项目
git clone https://github.com/Wechat-ggGitHub/wechat-claude-code.git ~/.claude/skills/wechat-claude-code

# 2. 进入目录
cd ~/.claude/skills/wechat-claude-code

# 3. 安装依赖
npm install

# 4. 首次设置
npm run setup  # 会生成二维码，用微信扫码绑定

# 5. 启动服务
npm run daemon -- start
```

### **数据存储位置**
```
~/.wechat-claude-code/
├── accounts/     # 微信账户凭证
├── config.env    # 全局配置
├── sessions/     # 会话数据
└── logs/         # 日志文件
```

## 🔒 **安全与隐私**

### **安全特性**
- ✅ **本地运行** - Claude Code在本地运行，数据不出本地
- ✅ **权限控制** - 可控制Claude能执行的操作
- ✅ **扫码绑定** - 使用个人微信，需要授权
- ✅ **MIT许可证** - 开源透明

### **隐私考虑**
- **聊天记录**: 存储在本地，不发送到云端
- **微信绑定**: 仅用于消息转发，不收集个人信息
- **权限审批**: 每个敏感操作都需要用户确认

### **潜在风险**
- **微信账号安全** - 绑定后可能被滥用
- **本地权限** - Claude Code有访问本地文件的能力
- **网络连接** - 需要保持网络连接
