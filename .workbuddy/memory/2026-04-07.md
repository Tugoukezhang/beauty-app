### 关键技术点：
- `npx skills add` 需要交互式终端(TTY)，在非交互环境中无法使用
- Python 脚本通过下载 GitHub repo zip 并提取含 SKILL.md 的目录来安装
- 不同仓库的 SKILL.md 位置不同：有的在根目录，有的在 `skills/` 子目录，有的在 `.claude/skills/` 等更深层
- 用深层搜索（max_depth=5）解决了这个问题
- 使用 `ssl.CERT_NONE` 解决了 SSL 证书问题
