## 🚀 如何使用系统

### 1. 日常使用
```bash
# 运行每日自检
python skills_self_check.py check

# 查看统计信息
python skills_self_check.py stats

# 搜索和更新 Skills
python skills_search_manager.py update
```

### 2. 批处理脚本
```bash
# 双击运行日常检查
daily_skills_check.bat

# 更新所有 Skills
update_skills.bat
```

- **每日 09:00** - 自动签到

## 🔍 详细功能说明

### 1. Skills 自检功能
- **完整性检查**: 验证 Skills 目录结构和文件完整性
- **版本跟踪**: 记录 Skills 版本变更历史
- **依赖检查**: 检查 Skills 的依赖关系
- **记忆存储**: 使用 SQLite 数据库永久保存记录
- **报告生成**: 生成 TXT 格式详细报告

### 2. Skills 搜索和更新功能
- **市场搜索**: 自动搜索 marketplace 中的 Skills
- **智能分类**: 自动识别前端/后端/移动端 Skills
- **自动安装**: 从 marketplace 安装新发现的 Skills
- **自动更新**: 检查并更新已安装 Skills 到最新版本
- **分类报告**: 生成 JSON 格式分类报告

### 3. 跨设备一致性管理
- **设备注册**: 自动注册当前设备信息
- **快照创建**: 创建 Skills 状态快照
- **一致性检查**: 比较不同设备间的 Skills 状态
- **差异报告**: 生成设备间差异报告
- **同步建议**: 提供同步和修复建议
