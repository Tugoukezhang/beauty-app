# i妆 - 美妆教学小程序

> 专业美妆教学平台

## 项目说明

本项目为i妆美妆小程序，基于微信小程序开发。

### 功能模块

- **首页**：推荐课程、热门帖子
- **课程**：课程分类、详情、购买、学习
- **社区**：帖子发布、点赞评论、话题
- **个人中心**：用户信息、订单、会员、签到

### 技术栈

- 微信小程序
- TDesign组件库
- 原生JavaScript

### 目录结构

```
beauty-app-main/
├── src/
│   ├── app.js          # 应用入口
│   ├── app.json        # 应用配置
│   ├── app.wxss        # 全局样式
│   ├── api/            # API接口
│   ├── utils/          # 工具函数
│   ├── assets/         # 静态资源
│   ├── pages/          # 页面
│   └── components/     # 组件
├── project.config.json # 项目配置
└── sitemap.json        # sitemap
```
