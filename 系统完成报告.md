## 🛠️ 如何使用系统

### 1. 日常维护（最简单方式）
```bash
# 双击运行以下批处理文件：
daily_skills_check.bat      # 运行每日检查
update_skills.bat           # 更新Skills
test_skills_system.bat      # 测试系统
```

### 2. 手动运行Python脚本
```bash
# 签到系统
python auto_signin.py
python check_reports.py

# Skills管理
python skills_self_check.py check       # 自检
python skills_self_check.py stats       # 统计
python skills_search_manager.py         # 搜索更新
python skills_manager.py register       # 设备注册
python skills_manager.py snapshot       # 创建快照
```

### 3. 查看报告
```bash
# 查看签到报告
python check_reports.py

# 查看Skills报告
type "C:\Users\lintianhao\.workbuddy\skills_reports\skills_self_check_daily_*.txt"
```
