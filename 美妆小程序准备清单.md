## 一、微信小程序账号配置

### 1. 注册小程序
- 访问 [微信公众平台](https://mp.weixin.qq.com/)
- 选择"小程序"注册
- 完成主体认证

### 2. 获取 AppID
- 登录后进入「开发」→「开发管理」→「开发设置」
- 复制你的 AppID

### 3. 修改配置
文件：`project.config.json`
```json
"appid": "你的AppID",  // 第46行
```

---

## 二、服务器配置

### 1. 购买服务器
推荐配置：
- CPU：2核
- 内存：4GB
- 带宽：5Mbps
- 系统：Ubuntu 22.04 / CentOS 7

### 2. API地址配置
文件：`src/app.js`
```javascript
baseUrl: 'https://你的服务器IP或域名',  // 第14行
ossUrl: 'https://你的OSS地址',         // 第15行
```

文件：`src/api/request.js`
```javascript
baseUrl: 'https://你的服务器IP或域名',  // 第10行
```

## 五、后端API接口（需要开发）

### 已预留接口
```
用户模块：
POST /api/user/sendCode     - 发送验证码
POST /api/user/login        - 手机号登录
POST /api/user/register     - 注册
GET  /api/user/info         - 获取用户信息

课程模块：
GET  /api/course/list       - 课程列表
GET  /api/course/detail     - 课程详情
GET  /api/course/category   - 课程分类

社区模块：
GET  /api/post/list          - 帖子列表
GET  /api/post/detail        - 帖子详情
POST /api/post/create        - 发布帖子
POST /api/post/comment       - 评论帖子
POST /api/post/like          - 点赞

文件上传：
POST /api/upload/image       - 上传图片
POST /api/upload/video       - 上传视频
```
